## 一、系統簡介

本系統為「升國中數理資優班甄試」線上考試平台，提供以下核心功能：

| 功能 | 說明 |
|------|------|
| 📚 題庫管理 | 建立、編輯、刪除各科目題目（選擇題、填充題、計算題） |
| 📝 試卷管理 | 組合試卷、設定時間與狀態（草稿／開放／關閉） |
| 🎓 線上作答 | 學生計時作答，系統自動評分 |
| 📊 成績統計 | 查看個人成績、全班排行、錯誤率分析 |

**支援科目：**
- **升國中（資優班）：** 數學、自然科學、物理、化學、生物、地球科學
- **國小六年級：** 數學、自然科學、自然、國語、英語、社會（及上述所有科目）

**題目類型：** 選擇題（自動評分）、填充題（自動評分）、計算題（預留人工批改）  
**難度等級：** 1–5 星（入門 → 競賽級）  
**學段分類：** 升國中（資優班）、國小六年級

## 二、安裝與啟動

### 首次安裝（只需執行一次）

1. 確認已安裝 [Node.js](https://nodejs.org/) 16 以上版本
2. 在系統資料夾中，雙擊執行 `setup.bat`

```
setup.bat 會自動執行：
  [1/4] 安裝 Node.js 套件（npm install）
  [2/4] 產生前端頁面
  [3/4] 初始化資料庫
  [4/4] 植入範例題目
```

3. 出現「設定完成！請執行 start.bat 啟動系統」即表示安裝成功

### 日常啟動

雙擊 `start.bat`，出現以下訊息即表示啟動成功：

```
🎓 升國中數理資優班考題系統
   http://localhost:3000
```

開啟瀏覽器，前往 **http://localhost:3000** 即可使用。

> ⚠️ **注意：** 伺服器視窗不可關閉，關閉後系統停止運作。

### 關閉系統

直接關閉 `start.bat` 所開啟的命令提示字元視窗即可。

## 四、學生端操作

### 4.1 選擇考試

1. 點選首頁「**📝 參加考試**」
2. 進入考試列表頁，可看到目前**開放中**的試卷
3. 每張試卷卡片顯示：
   - 試卷名稱與說明
   - 考試時間（分鐘）
   - 題目數量與總分
4. 點選欲參加的試卷，進入考生資料填寫頁

> ℹ️ 只有狀態為「**開放中（active）**」的試卷才會顯示。草稿或已關閉的試卷不會出現。

---

### 4.2 填寫考生資料

進入試卷後，需填寫以下資料：

| 欄位 | 必填 | 說明 |
|------|------|------|
| **姓名** | ✅ 必填 | 請輸入真實姓名，成績單將顯示此名稱 |
| **座號／學號** | 選填 | 可填入班級座號或學號以便查詢 |

填寫完畢後，點選「**開始考試**」按鈕進入作答畫面。

---

### 4.3 進行作答

進入作答畫面後，介面包含以下區域：

```
┌──────────────────────────────────────────┐
│  試卷標題        剩餘時間 [MM:SS] [提交]  │
├──────────────────────────────────────────┤
│                                          │
│  第 1 題（選擇題）                        │
│  題目內容文字...                          │
│                                          │
│  ○ A. 選項一                             │
│  ○ B. 選項二                             │
│  ○ C. 選項三                             │
│  ○ D. 選項四                             │
│                                          │
│  第 2 題（填充題）                        │
│  題目內容... ＿＿＿                       │
│  [請填寫答案                          ]  │
│                                          │
├──────────────────────────────────────────┤
│  題目快速導覽：[1][2][3][4]...[N]        │
└──────────────────────────────────────────┘
```

#### 作答說明

| 題目類型 | 操作方式 |
|----------|----------|
| **選擇題** | 點選 A / B / C / D 選項的圓形按鈕 |
| **填充題** | 在輸入框中直接輸入答案文字 |
| **計算題** | 在輸入框中輸入計算結果 |

#### 計時器

- 右上角顯示**倒數計時**（格式：MM:SS）
- 時間到 **0:00** 時系統自動提交，無需手動操作
- 建議在時間結束前確認作答完整性

#### 題目導覽列

- 畫面下方顯示所有題號按鈕
- 點選題號可快速跳至該題
- 已作答的題目按鈕會改變顯示樣式

---

### 4.4 提交與查看成績

#### 手動提交

點選右上角「**提交作答**」→ 確認提示視窗 → 點選「**確定**」

> ⚠️ 提交後無法修改，請確認作答完畢再提交。

#### 成績頁面

提交後自動跳轉至個人成績頁，顯示：

```
┌──────────────────────────────┐
│      王小明 的成績            │
│                              │
│           85%                │
│   85 / 100 分 · 作答時間     │
│                              │
│  ✅ 優秀！繼續保持！          │
└──────────────────────────────┘
```

- **百分比顯示**：依得分比例以顏色標示（綠色：優秀 / 黃色：良好 / 橘色：加油）
- **每題詳解**：顯示每一題的題目、你的答案、正確答案及詳細解析
- 標記 ✓ **答對**（+得分）或 ✗ **答錯**

#### 成績頁面網址格式

```
http://localhost:3000/result.html?id=<作答ID>
```

可將此網址分享給家長查看成績。

## 五、教師端操作（後台管理）

### 5.1 進入後台

點選首頁「**⚙️ 後台管理**」，或直接前往：

```
http://localhost:3000/admin.html
```

後台分為三個功能頁籤：

| 頁籤 | 功能 |
|------|------|
| **題庫管理** | 新增、編輯、刪除題目 |
| **試卷管理** | 建立試卷、組合題目、控制開放狀態 |
| **成績統計** | 查看各試卷統計與錯誤率分析 |

---

### 5.2 題庫管理

點選上方「**題庫管理**」頁籤（預設顯示）。

#### 篩選與搜尋

頁面上方提供五種篩選條件，可組合使用：

| 篩選項目 | 說明 |
|----------|------|
| **全部科目** 下拉選單 | 篩選特定科目（數學、物理…） |
| **全部類型** 下拉選單 | 篩選選擇題 / 填充題 / 計算題 |
| **全部難度** 下拉選單 | 篩選 1–5 星難度 |
| **全部學段** 下拉選單 | 篩選「國小六年級」或「升國中（資優班）」 |
| **搜尋框** | 輸入關鍵字即時搜尋題目內容或標籤 |

#### 新增題目

1. 點選右上角「**＋ 新增題目**」按鈕
2. 填寫題目表單：

| 欄位 | 必填 | 說明 |
|------|------|------|
| **科目** | ✅ | 選擇題目所屬科目 |
| **題型** | ✅ | 選擇題 / 填充題 / 計算題 |
| **難度** | ✅ | 1（入門）至 5（競賽級） |
| **學段** | ✅ | 升國中（資優班） / 國小六年級（預設：升國中） |
| **題目內容** | ✅ | 輸入題目文字，可包含數學符號 |
| **選項 A/B/C/D** | 選擇題必填 | 填入各選項文字 |
| **正確答案** | ✅ | 選擇題填 A/B/C/D；填充題填正確答案文字 |
| **解析** | 選填 | 詳細解題說明，作答後顯示給學生 |
| **來源** | 選填 | 題目出處（如：XX年份考題） |
| **標籤** | 選填 | 以逗號分隔，方便搜尋（如：數論,質數） |

3. 點選「**儲存**」完成新增

#### 編輯題目

在題目列表中，點選該題右側的「**編輯**」按鈕，修改後點選「**儲存**」。

#### 刪除題目

在題目列表中，點選該題右側的「**刪除**」按鈕，確認後刪除。

> ⚠️ 刪除後無法復原，請確認該題未被試卷使用。

#### 題目列表欄位說明

| 欄位 | 說明 |
|------|------|
| **ID** | 系統自動編號 |
| **科目** | 題目所屬科目 |
| **類型** | choice（選擇）/ fill（填充）/ calculation（計算） |
| **難度** | ★ 至 ★★★★★ |
| **學段** | 🟢 國小六年級 / 🔵 升國中（資優班） |
| **題目** | 題目內容前段（截斷顯示） |
| **標籤** | 分類標籤 |
| **操作** | 編輯 / 刪除按鈕 |

### 5.3 試卷管理

點選上方「**試卷管理**」頁籤。

#### 新增試卷

1. 點選右上角「**＋ 新增試卷**」按鈕
2. 填寫試卷基本資訊：

| 欄位 | 必填 | 說明 |
|------|------|------|
| **試卷名稱** | ✅ | 如：112學年度數理資優班入學測驗 |
| **說明** | 選填 | 考試注意事項或說明文字 |
| **考試時間（分鐘）** | ✅ | 預設 90 分鐘 |
| **狀態** | ✅ | 草稿 / 開放中 / 已關閉 |

3. **選題（從1,000題中挑選）**，詳見下方說明
4. 點選「**儲存**」完成建立

#### 更換試卷題目

無論是新增或**編輯現有試卷**，皆可透過選題區域完整更換題目。

**選題區域介面說明：**

```
┌─────────────────────────────────────────────────────────────────────┐
│ 選擇題目        [全部科目 ▼]  [全部學段 ▼]  [全部難度 ▼]           │
├─────────────────────────────────────────────────────────────────────┤
│ ☑ 若 $x^2-5x+6=0$，則 x = ？          數學 ★★★                   │
│ ☐ 下列何者為質數？                      數學 ★★                    │
│ ☐ 細胞膜的主要成分是？                  生物 ★★                    │
│ ...（最多顯示 100 題）                                               │
└─────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────┐
│ 🎲 隨機抽題                                                          │
│ 科目[全部▼]  題型[全部▼]  最低難度[不限▼]  最高難度[不限▼]         │
│ 抽題數量[10]                        [隨機抽題]                       │
│ 已加入 10 題（共抽到 10 題，其中 0 題已在清單中）                    │
└─────────────────────────────────────────────────────────────────────┘
```

| 功能 | 操作方式 | 說明 |
|------|----------|------|
| **科目篩選** | 下拉選單選擇科目 | 只顯示該科目題目 |
| **學段篩選** | 下拉選單選擇學段 | 只顯示「國小六年級」或「升國中」題目 |
| **難度篩選** | 下拉選單選擇星級 | 只顯示該難度題目 |
| **勾選加入** | 點選題目左側核取方塊 | 打勾即加入「已選題目」清單 |
| **移除題目** | 點選已選清單右側的 ✕ | 從試卷中移除該題 |
| **個別配分** | 在已選清單輸入分值 | 每題預設 5 分，可個別調整 |

> 💡 **小技巧：** 科目、學段與難度篩選可組合使用。例如：選「國小六年級」+「數學」+「★★★」可快速預覽國小中級數學題；搭配下方 🎲 隨機抽題更可進一步指定題型與難度上下限，一鍵完成出卷。

#### 隨機抽題

此功能可依條件自動從題庫中隨機選取指定數量的題目，適合快速組卷。試卷編輯畫面中有獨立的「**🎲 隨機抽題**」區塊，位於「選擇題目」清單下方。

**操作步驟：**

1. （選用）在 🎲 隨機抽題區塊設定篩選條件：

| 篩選欄位 | 說明 |
|----------|------|
| **科目** | 只從指定科目抽題（不選則全部科目） |
| **題型** | 限定選擇題 / 填充題 / 計算題（不選則全部） |
| **最低難度** | 難度下限（★ ～ ★★★★★，不限則不設下限） |
| **最高難度** | 難度上限（★ ～ ★★★★★，不限則不設上限） |
| **抽題數量** | 預設 10 題，最多 100 題 |

2. 點選「**隨機抽題**」按鈕
3. 系統自動從符合條件的題目中隨機選取，加入「已選題目」清單
4. 面板下方顯示本次結果，例如：「已加入 10 題（共抽到 10 題，其中 0 題已在清單中）」
5. 確認後點選「**儲存**」

> ⚠️ **注意：**
> - 隨機抽題不會重複加入已選的題目
> - 每次點擊「隨機抽題」會**累加**至已選清單（不會覆蓋現有選題）
> - 若要全部重新抽題，請先在已選清單逐一移除，再執行隨機抽題
> - 若符合條件的題目不足指定數量，系統將返回全部符合的題目

**組合使用範例（組合式出卷建議）：**

| 步驟 | 操作 | 效果 |
|------|------|------|
| 1 | 科目選「數學」，最低難度「★★★」，抽 10 題 | 加入 10 題數學中高級題 |
| 2 | 科目選「物理」，題型選「選擇題」，抽 5 題 | 再加入 5 題物理選擇題 |
| 3 | 科目選「化學」，難度不限，抽 5 題 | 再加入 5 題化學題 |
| 結果 | 共 20 題，涵蓋三科不同類型 | |

#### 試卷狀態說明

| 狀態 | 說明 | 學生可見 |
|------|------|----------|
| **草稿（draft）** | 尚未完成，不對外開放 | ❌ |
| **開放中（active）** | 學生可選擇並作答 | ✅ |
| **已關閉（closed）** | 考試結束，不再接受作答 | ❌ |

#### 控制試卷開放

1. 在試卷列表中點選「**編輯**」
2. 修改「狀態」欄位：
   - 開始考試前：設為「**草稿**」→ 準備好後改為「**開放中**」
   - 考試結束後：設為「**已關閉**」避免繼續作答
3. 點選「**儲存**」

#### 試卷列表欄位說明

| 欄位 | 說明 |
|------|------|
| **試卷名稱** | 試卷標題與說明 |
| **題數** | 試卷包含題目數量 |
| **總分** | 所有題目配分加總 |
| **狀態** | 目前開放狀態 |
| **操作** | 編輯 / 刪除按鈕 |

### 5.4 成績統計

點選上方「**成績統計**」頁籤。

1. 從下拉選單選擇欲查看的試卷
2. 頁面顯示該試卷統計資訊：

#### 統計摘要

| 統計項目 | 說明 |
|----------|------|
| **作答人數** | 提交作答的學生總數 |
| **平均分** | 全班平均百分比 |
| **最高分** | 最高得分百分比 |
| **最低分** | 最低得分百分比 |

#### 最常答錯題目 Top 5

顯示全班答錯次數最多的 5 道題目，可用於：
- 找出教學盲點
- 決定是否需要課堂複習

#### 個別成績列表

點選「**成績列表**」頁籤可查看個別學生成績，欄位包含：

| 欄位 | 說明 |
|------|------|
| **姓名** | 學生填寫的考生姓名 |
| **學號** | 學生填寫的座號（若有填寫） |

| **百分比** | 得分換算百分比 |

## 七、系統維護

### 補充題庫

如需批次補充題目，可執行以下指令：

```bat
node seed.js
```

此指令會新增範例數學與自然科學題目，並建立一份範例試卷。

### 資料庫備份

資料庫檔案位於系統資料夾中的 `exam.db`。

**備份方式：**

1. 關閉伺服器（關閉 start.bat 視窗）
2. 複製 `exam.db` 到安全位置
3. 重新啟動伺服器

> ⚠️ 請勿在伺服器運作中直接複製資料庫，可能導致資料不完整。

**使用 DB Browser 查看資料庫：**

下載 [DB Browser for SQLite](https://sqlitebrowser.org/)，開啟 `exam.db` 可直接瀏覽和匯出資料。

### 重新產生前端頁面

若前端頁面遺失或損壞：

```bat
node generate-public.js
```

### 自訂連接埠

預設使用 **3000** 埠。若需更改：

```bat
set PORT=8080 && node server.js
```

## 八、常見問題

**Q1：學生看不到任何試卷？**  
→ 確認試卷狀態已設為「**開放中（active）**」。草稿與已關閉的試卷不會顯示。

**Q2：伺服器無法啟動，出現「address already in use」？**  
→ 埠 3000 已被其他程式佔用。請關閉其他使用 3000 埠的程式，或以 `set PORT=3001 && node server.js` 使用其他埠。

**Q3：填充題答案輸入後沒有得分？**  
→ 系統以**完全比對**方式評分（不區分大小寫）。請確認正確答案設定時的格式與學生可能輸入的格式一致。例如：答案設定為 `H₂O`，學生若輸入 `h2o` 則不符。建議在解析欄說明標準答案格式。

**Q4：如何刪除學生的作答紀錄？**  
→ 目前系統不提供前端介面刪除單筆作答紀錄。如有需要，可使用 [DB Browser for SQLite](https://sqlitebrowser.org/) 開啟 `exam.db`，在 `submissions` 資料表中手動刪除。

**Q5：考試時間到了學生還沒提交會怎樣？**  
→ 倒數歸零後系統會**自動提交**目前的作答內容，學生無需手動操作。

**Q6：可以讓同一位學生重複作答嗎？**  
→ 可以。系統允許同一姓名多次提交，每次均視為獨立作答紀錄，成績列表中會顯示所有紀錄。

**Q7：題庫有多少題？**  
→ 系統目前內建 **1,076 道題目**（升國中資優班），涵蓋 6 個科目。國小六年級題庫需由教師自行新增。教師可透過後台持續新增題目。

**Q8：如何匯出成績報表？**  
→ 目前系統不內建匯出功能。可開啟瀏覽器的「**另存為 HTML**」儲存成績頁面，或使用 DB Browser 將 `submissions` 資料表匯出為 CSV 檔案。

**Q9：如何新增國小六年級題目？**  
→ 進入後台管理 → 題庫管理 → 點選「**＋ 新增題目**」→ 在「**學段**」欄位選擇「**國小六年級**」，科目可選擇數學、**自然**、自然科學、國語、英語或社會。

**Q10：如何只看某一學段的題目？**  
→ 在後台管理「題庫管理」頁面，點選「**全部學段**」下拉選單，選擇「國小六年級」或「升國中」即可篩選顯示。

**Q11：組卷時可以混合兩個學段的題目嗎？**  
→ 可以。試卷選題時不限制學段，可同時勾選國小六年級與升國中的題目組成同一份試卷。若要分開出卷，在選題區的「全部學段」下拉選單選擇特定學段即可快速篩選。
